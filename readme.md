# 表白墙（love-love-wall）项目

## 1、环境

react脚手架，包含：

* antd
* axios
* autoprefixer（增加css兼容性后缀）
* babel
* eslint
* less
* postcss
* webpack3
* uglifyjs-webpack-plugin（代码压缩混淆）

React全家桶里只有React本身

多入口放在 src/page 目录下，见文件注释和示例DEMO

---

使用 Antd 作为 UI ，参考链接：

https://ant.design/docs/react/getting-started-cn
